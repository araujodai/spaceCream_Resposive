# Space Cream

Quarto desafio proposto durante o Stage 03 da trilha Explorer com o layout disponibilizado pela Rocketseat.
O desafio consiste em reproduzir o layout, com mobile first, utilizando:

- Grid;
- Animações e transições;
- Media queries.

## Construído com:

- HTML
- CSS
